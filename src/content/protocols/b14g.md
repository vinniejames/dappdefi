b14g is a modular dual-staking layer for Bitcoin. Participating networks let validators secure consensus with both BTC and the network's native token, reducing the sell-pressure path and pairing native security with Bitcoin's economic weight.

The protocol integrates with CORE Chain and Babylon Genesis as primary execution surfaces, and is one of several projects exploring how Bitcoin-secured PoS can extend beyond pure restaking into more native dual-staking architectures.
